# الفصل 1: تسجيل الدخول

*الجمهور: الجميع*

## طرق المصادقة

يدعم Cat-Scan ثلاث طرق لتسجيل الدخول:

| الطريقة | كيف تعمل | متى تُستخدم |
|---------|----------|-------------|
| **Google OAuth** | انقر على "تسجيل الدخول بحساب Google"، والذي يعيد التوجيه عبر OAuth2 Proxy | معظم المستخدمين. يستخدم حساب Google Workspace الخاص بك. |
| **Authing (OIDC)** | انقر على "تسجيل الدخول بـ Authing"، والذي يعيد التوجيه إلى موفر OIDC | المؤسسات التي تستخدم Authing كموفر هوية. |
| **البريد الإلكتروني وكلمة المرور** | أدخل بيانات الاعتماد مباشرة في صفحة تسجيل الدخول | الحسابات المحلية التي أنشأها المسؤول. |

## تسجيل الدخول لأول مرة

1. انتقل إلى `https://scan.rtb.cat` (أو عنوان URL الخاص بنشرك).
2. ستظهر لك صفحة تسجيل الدخول مع خيارات الدخول المتاحة.
3. اختر طريقتك وقم بالمصادقة.
4. عند تسجيل الدخول لأول مرة، ينشئ النظام سجل المستخدم الخاص بك تلقائيًا
   (لطرق OAuth). قد يحتاج المسؤول إلى منحك صلاحية الوصول إلى مقاعد
   مشترين محددة.

## محدد المقعد

بعد تسجيل الدخول، سترى الشريط الجانبي مع **محدد المقعد** في الأعلى.
إذا كان حسابك يمتلك صلاحية الوصول إلى عدة مقاعد مشترين، استخدم القائمة
المنسدلة للتبديل بينها. جميع البيانات في كل صفحة محددة النطاق بالمقعد المحدد.

- **مقعد واحد**: يعرض المحدد اسم المقعد والمعرّف مباشرة.
- **مقاعد متعددة**: تتيح لك القائمة المنسدلة التبديل. يعرض كل إدخال الاسم
  المعروض للمشتري، و`buyer_account_id`، وعدد التصميمات الإبداعية.
- **زر "مزامنة الكل"**: يحدّث التصميمات الإبداعية ونقاط النهاية وإعدادات
  الاستهداف المسبق من API الخاص بـ Google للمقعد المحدد.

## عند فشل تسجيل الدخول

| العرَض | السبب المحتمل | ما يجب فعله |
|--------|--------------|-------------|
| حلقة إعادة توجيه (الصفحة تستمر في إعادة التحميل) | قاعدة البيانات غير قابلة للوصول، مما يؤدي إلى فشل فحص المصادقة بصمت | تحقق من حاوية Cloud SQL Proxy. انظر [استكشاف الأخطاء وإصلاحها](15-troubleshooting.md). |
| "الخادم غير متاح" (502/503/504) | حاوية API أو nginx متوقفة | تواصل مع فريق DevOps الخاص بك. انظر [مراقبة الصحة](13-health-monitoring.md). |
| "المصادقة مطلوبة" | انتهت صلاحية الجلسة أو تم مسح ملف تعريف الارتباط | سجّل الدخول مرة أخرى. |
| "ليس لديك صلاحية الوصول إلى حساب المشتري هذا" | لم يتم منح الصلاحيات لهذا المقعد | اسأل المسؤول الخاص بك. انظر [إدارة المستخدمين](16-user-admin.md). |
