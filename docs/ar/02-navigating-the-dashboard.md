# الفصل 2: التنقل في لوحة المعلومات

*الجمهور: الجميع*

## تخطيط الشريط الجانبي

الشريط الجانبي هو وسيلة التنقل الأساسية. يمكن طيّه (وضع الأيقونات فقط)
أو توسيعه. يتم حفظ تفضيلك عبر الجلسات.

```
Seat Selector
 ├── QPS Waste Optimizer         /              (home)
 ├── Creatives                   /creatives
 ├── Campaigns                   /campaigns
 ├── Change History              /history
 ├── Import                      /import
 │
 ├── QPS (expandable)
 │   ├── Publisher                /qps/publisher
 │   ├── Geo                     /qps/geo
 │   └── Size                    /qps/size
 │
 ├── Settings (expandable)
 │   ├── Connected Accounts      /settings/accounts
 │   ├── Data Retention          /settings/retention
 │   └── System Status           /settings/system
 │
 ├── Admin (sudo users only)
 │   ├── Users                   /admin/users
 │   ├── Configuration           /admin/configuration
 │   └── Audit Log               /admin/audit-log
 │
 └── Footer: user email, version, docs link
```

تتوسع الأقسام تلقائيًا عند التنقل إليها.

## المستخدمون المقيّدون

تُعلَّم بعض الحسابات على أنها "مقيّدة" من قبل المسؤول. يرى المستخدمون
المقيّدون الصفحات الأساسية فقط: الرئيسية، التصميمات الإبداعية، الحملات،
الاستيراد، والسجل. يتم إخفاء أقسام تحليل QPS والإعدادات والمسؤول.

## قائمة الإعداد

تظهر للحسابات الجديدة قائمة إعداد في `/setup` تُرشدك خلال الإعداد الأولي:

1. ربط حسابات المشترين (رفع بيانات اعتماد GCP، واكتشاف المقاعد)
2. التحقق من صحة البيانات (التأكد من وصول استيرادات CSV)
3. تسجيل نموذج محسّن (نقطة نهاية BYOM)
4. التحقق من نقطة نهاية النموذج (استدعاء اختباري)
5. تحديد خط أساس تكلفة الاستضافة (لحسابات الاقتصاديات)
6. ربط مصدر تحويلات (بيكسل أو webhook)

يتم تتبع نسبة الإكمال. تربط كل خطوة بصفحة الإعدادات ذات الصلة.

## دعم اللغات

يدعم Cat-Scan اللغات الإنجليزية والهولندية والصينية (المبسطة). يوجد محدد
اللغة في الشريط الجانبي. يتم حفظ تفضيلك لكل مستخدم.
