# Глава 2: Навигация по панели управления

*Аудитория: все пользователи*

## Структура боковой панели

Боковая панель — основной элемент навигации. Её можно свернуть (режим иконок)
или развернуть. Ваши предпочтения сохраняются между сессиями.

```
Seat Selector
 ├── QPS Waste Optimizer         /              (home)
 ├── Creatives                   /creatives
 ├── Campaigns                   /campaigns
 ├── Change History              /history
 ├── Import                      /import
 │
 ├── QPS (expandable)
 │   ├── Publisher                /qps/publisher
 │   ├── Geo                     /qps/geo
 │   └── Size                    /qps/size
 │
 ├── Settings (expandable)
 │   ├── Connected Accounts      /settings/accounts
 │   ├── Data Retention          /settings/retention
 │   └── System Status           /settings/system
 │
 ├── Admin (sudo users only)
 │   ├── Users                   /admin/users
 │   ├── Configuration           /admin/configuration
 │   └── Audit Log               /admin/audit-log
 │
 └── Footer: user email, version, docs link
```

Разделы автоматически раскрываются при переходе к ним.

## Пользователи с ограниченным доступом

Некоторые аккаунты отмечены администратором как «ограниченные» (restricted).
Такие пользователи видят только основные страницы: главная, креативы, кампании,
импорт и история. Разделы QPS-анализа, настроек и администрирования скрыты.

## Контрольный список настройки

Новые аккаунты видят контрольный список настройки по адресу `/setup`, который
проведёт через начальную конфигурацию:

1. Подключение аккаунтов покупателя (загрузка учётных данных GCP, обнаружение мест)
2. Проверка состояния данных (проверка поступления CSV-импортов)
3. Регистрация модели оптимизатора (BYOM-эндпоинт)
4. Проверка эндпоинта модели (тестовый вызов)
5. Установка базовых затрат на хостинг (для расчёта экономики)
6. Подключение источника конверсий (пиксель или вебхук)

Процент выполнения отслеживается. Каждый шаг содержит ссылку на соответствующую
страницу настроек.

## Поддержка языков

Cat-Scan поддерживает английский, нидерландский и китайский (упрощённый).
Переключатель языка находится в боковой панели. Ваш выбор сохраняется для
каждого пользователя.
