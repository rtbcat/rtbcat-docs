# Розділ 2: Навігація панеллю керування

*Аудиторія: всі*

## Структура бічної панелі

Бічна панель — це ваша основна навігація. Її можна згорнути (режим лише
іконок) або розгорнути. Ваші вподобання зберігаються між сесіями.

```
Seat Selector
 ├── QPS Waste Optimizer         /              (home)
 ├── Creatives                   /creatives
 ├── Campaigns                   /campaigns
 ├── Change History              /history
 ├── Import                      /import
 │
 ├── QPS (expandable)
 │   ├── Publisher                /qps/publisher
 │   ├── Geo                     /qps/geo
 │   └── Size                    /qps/size
 │
 ├── Settings (expandable)
 │   ├── Connected Accounts      /settings/accounts
 │   ├── Data Retention          /settings/retention
 │   └── System Status           /settings/system
 │
 ├── Admin (sudo users only)
 │   ├── Users                   /admin/users
 │   ├── Configuration           /admin/configuration
 │   └── Audit Log               /admin/audit-log
 │
 └── Footer: user email, version, docs link
```

Секції автоматично розгортаються, коли ви переходите до них.

## Обмежені користувачі

Деякі облікові записи позначені адміністратором як «обмежені». Обмежені
користувачі бачать лише основні сторінки: головну, креативи, кампанії, імпорт
та історію. Секції аналізу QPS, налаштувань та адміністрування приховані.

## Контрольний список налаштування

Нові облікові записи бачать контрольний список налаштування за адресою `/setup`,
який проведе вас через початкову конфігурацію:

1. Підключення облікових записів покупців (завантаження облікових даних GCP, виявлення seat-ів)
2. Перевірка стану даних (перевірка надходження CSV-імпортів)
3. Реєстрація моделі оптимізатора (BYOM endpoint)
4. Валідація endpoint-у моделі (тестовий виклик)
5. Встановлення базових витрат на хостинг (для розрахунків економіки)
6. Підключення джерела конверсій (піксель або webhook)

Відсоток виконання відстежується. Кожен крок посилається на відповідну
сторінку налаштувань.

## Підтримка мов

Cat-Scan підтримує англійську, нідерландську та китайську (спрощену). Вибір
мови знаходиться на бічній панелі. Ваші вподобання зберігаються для кожного
користувача.
