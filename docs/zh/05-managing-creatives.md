# 第 5 章：管理素材

*适用读者：媒体买手、投放经理*

## 素材库（`/creatives`）

素材库展示与所选席位关联的所有素材。

![素材库，展示格式标签和表现层级](../images/screenshot-creatives.png)

### 你看到什么

每个素材以卡片形式展示：

- **缩略图**：自动生成的广告预览（视频帧或展示截图）
- **格式标签**：VIDEO、DISPLAY_IMAGE、DISPLAY_HTML 或 NATIVE
- **素材 ID**：Authorized Buyers 素材标识符
- **标准尺寸**：主要广告尺寸（如 300x250、728x90）
- **表现层级**：HIGH、MEDIUM、LOW 或 NO_DATA，基于席位内花费百分位排名

### 筛选和搜索

- **格式筛选器**：仅显示视频、展示图片、展示 HTML 或原生广告
- **表现层级筛选器**：隔离高表现或低表现素材
- **搜索**：按 ID 查找素材
- **时间段选择器**：7、14 或 30 天的效果数据

### 缩略图

缩略图按批次生成。如果看到占位图片，使用批量生成缩略图按钮排队生成缺失的
缩略图。状态会在界面中显示。

### 素材详情

点击任何素材打开预览弹窗，包含：

- 目标页 URL 和诊断（着陆页是否可达？）
- 语言检测（自动检测 + 手动覆盖选项）
- 国家表现细分（此素材在哪些地区表现好）
- 语言-地理报告（语言与地理位置不匹配检测）

**语言不匹配检测**是一个特色功能：Cat-Scan 可以标记诸如西班牙语广告在
阿拉伯市场投放、或 AED 定价定向印度用户等情况。这使用你配置的 AI 供应商
（Gemini、Claude 或 Grok）。

## 活动聚类（`/campaigns`）

活动让你将素材组织成逻辑分组。

### 视图

- **网格视图**：活动卡片，包含素材数量、花费、展示、点击
- **列表视图**：紧凑的表格格式

### 操作

- **拖放**：在活动之间或移入未分配池中移动素材
- **创建活动**：命名新分组并将素材拖入
- **AI 自动聚类**：让 Cat-Scan 根据素材属性（格式、尺寸、目标页、语言）
  建议分组
- **删除活动**：移除分组（素材回到未分配池）

### 筛选器

- **排序方式**：名称、花费、展示、点击、素材数量
- **国家筛选器**：仅显示在特定地区有投放素材的活动
- **问题筛选器**：高亮有问题的活动（不匹配、低表现者）
