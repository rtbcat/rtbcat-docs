*[QPS]: Запросы в секунду — частота запросов на ставки, которые Google отправляет вашему биддеру. Каждый запрос стоит вычислительных ресурсов и пропускной способности, поэтому потраченный впустую QPS — это потраченные впустую деньги.
*[RTB]: Торги в реальном времени — аукционный протокол, в котором рекламные показы покупаются и продаются за миллисекунды.
*[CPM]: Стоимость за тысячу — цена за 1 000 рекламных показов.
*[CTR]: Показатель кликабельности — клики, разделённые на показы, выраженные в процентах.
*[ROAS]: Рентабельность рекламных расходов — доход на каждый доллар, потраченный на рекламу.
*[BYOM]: Принеси свою модель — система Cat-Scan для подключения собственной внешней модели оценки для формирования предложений по оптимизации.
*[CSV]: Значения, разделённые запятыми — формат файлов, используемый Google Authorized Buyers для экспорта отчётов.
*[AB]: Authorized Buyers — программатическая рекламная платформа Google (ранее AdX / DoubleClick Ad Exchange).
*[HMAC]: Код аутентификации сообщений на основе хеша — криптографическая подпись для проверки полезных данных вебхуков.
*[SRE]: Инженерия надёжности сайтов — дисциплина эксплуатации надёжных производственных систем.
*[IAP]: Identity-Aware Proxy — контроль доступа с нулевым доверием от Google Cloud для SSH и HTTP сервисов.
*[DSN]: Имя источника данных — строка подключения к базе данных PostgreSQL.
