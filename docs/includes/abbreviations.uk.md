*[QPS]: Запити на секунду — частота запитів на ставки, які Google надсилає вашому бідеру. Кожен запит коштує обчислювальних ресурсів та пропускної здатності, тому марний QPS — це марні гроші.
*[RTB]: Торги в реальному часі — аукціонний протокол, де рекламні покази купуються та продаються за мілісекунди.
*[CPM]: Вартість за тисячу — ціна за 1 000 рекламних показів.
*[CTR]: Показник клікабельності — кліки, поділені на покази, виражені у відсотках.
*[ROAS]: Рентабельність рекламних витрат — дохід на кожен долар, витрачений на рекламу.
*[BYOM]: Принеси свою модель — система Cat-Scan для підключення власної зовнішньої моделі оцінювання для формування пропозицій з оптимізації.
*[CSV]: Значення, розділені комами — формат файлів, який Google Authorized Buyers використовує для експорту звітів.
*[AB]: Authorized Buyers — програматична рекламна платформа Google (раніше AdX / DoubleClick Ad Exchange).
*[HMAC]: Код автентифікації повідомлень на основі хешу — криптографічний підпис для перевірки даних вебхуків.
*[SRE]: Інженерія надійності сайтів — дисципліна експлуатації надійних виробничих систем.
*[IAP]: Identity-Aware Proxy — контроль доступу з нульовою довірою від Google Cloud для SSH та HTTP сервісів.
*[DSN]: Ім'я джерела даних — рядок підключення до бази даних PostgreSQL.
