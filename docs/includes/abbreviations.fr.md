*[QPS]: Requêtes par seconde — le débit de demandes d'enchères que Google envoie à votre enchérisseur. Chaque requête coûte en calcul et en bande passante, donc du QPS gaspillé est de l'argent gaspillé.
*[RTB]: Enchères en temps réel — le protocole d'enchères où les impressions publicitaires sont achetées et vendues en millisecondes.
*[CPM]: Coût pour mille — le prix payé pour 1 000 impressions publicitaires.
*[CTR]: Taux de clics — clics divisés par les impressions, exprimé en pourcentage.
*[ROAS]: Retour sur dépenses publicitaires — revenus générés par dollar dépensé en publicité.
*[BYOM]: Apportez votre propre modèle — le système de Cat-Scan pour connecter votre propre modèle de notation externe afin de générer des propositions d'optimisation.
*[CSV]: Valeurs séparées par des virgules — le format de fichier utilisé par Google Authorized Buyers pour l'exportation des rapports.
*[AB]: Authorized Buyers — la plateforme publicitaire programmatique de Google (anciennement AdX / DoubleClick Ad Exchange).
*[HMAC]: Code d'authentification de message basé sur le hachage — une signature cryptographique utilisée pour vérifier les charges utiles des webhooks.
*[SRE]: Ingénierie de la fiabilité des sites — la discipline de l'exploitation de systèmes de production fiables.
*[IAP]: Identity-Aware Proxy — le contrôle d'accès zero-trust de Google Cloud pour les services SSH et HTTP.
*[DSN]: Nom de source de données — la chaîne de connexion utilisée pour se connecter à une base de données PostgreSQL.
