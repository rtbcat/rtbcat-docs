*[QPS]: Consultas por segundo — la tasa de solicitudes de puja que Google envía a tu pujador. Cada solicitud cuesta procesamiento y ancho de banda, así que QPS desperdiciado es dinero desperdiciado.
*[RTB]: Puja en tiempo real — el protocolo de subasta donde las impresiones publicitarias se compran y venden en milisegundos.
*[CPM]: Coste por mil — el precio pagado por cada 1.000 impresiones publicitarias.
*[CTR]: Tasa de clics — clics divididos entre impresiones, expresado como porcentaje.
*[ROAS]: Retorno de la inversión publicitaria — ingresos generados por cada dólar gastado en publicidad.
*[BYOM]: Trae tu propio modelo — el sistema de Cat-Scan para conectar tu propio modelo de puntuación externo para generar propuestas de optimización.
*[CSV]: Valores separados por comas — el formato de archivo que Google Authorized Buyers usa para exportar informes.
*[AB]: Authorized Buyers — la plataforma de publicidad programática de Google (anteriormente AdX / DoubleClick Ad Exchange).
*[HMAC]: Código de autenticación de mensajes basado en hash — una firma criptográfica utilizada para verificar las cargas de webhook.
*[SRE]: Ingeniería de fiabilidad del sitio — la disciplina de operar sistemas de producción fiables.
*[IAP]: Identity-Aware Proxy — el control de acceso zero-trust de Google Cloud para servicios SSH y HTTP.
*[DSN]: Nombre de origen de datos — la cadena de conexión utilizada para conectarse a una base de datos PostgreSQL.
