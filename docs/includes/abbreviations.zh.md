*[QPS]: 每秒查询数 — Google 向您的竞价器发送出价请求的速率。每个请求都消耗计算和带宽资源，因此浪费的 QPS 就是浪费的金钱。
*[RTB]: 实时竞价 — 广告展示在毫秒内买卖的拍卖协议。
*[CPM]: 每千次展示费用 — 每 1,000 次广告展示支付的价格。
*[CTR]: 点击率 — 点击次数除以展示次数，以百分比表示。
*[ROAS]: 广告支出回报率 — 每花费一美元广告费产生的收入。
*[BYOM]: 自带模型 — Cat-Scan 的系统，用于接入您自己的外部评分模型以生成优化建议。
*[CSV]: 逗号分隔值 — Google Authorized Buyers 用于导出报告的文件格式。
*[AB]: Authorized Buyers — Google 的程序化广告交易平台（前身为 AdX / DoubleClick Ad Exchange）。
*[HMAC]: 基于哈希的消息认证码 — 用于验证 webhook 载荷的加密签名。
*[SRE]: 站点可靠性工程 — 运维可靠生产系统的学科。
*[IAP]: 身份感知代理 — Google Cloud 的零信任访问控制，用于 SSH 和 HTTP 服务。
*[DSN]: 数据源名称 — 用于连接 PostgreSQL 数据库的连接字符串。
