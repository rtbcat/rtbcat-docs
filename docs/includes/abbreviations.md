*[QPS]: Queries Per Second — the maximum rate of bid requests Google sends your bidder. You pay for this capacity whether you use it or not.
*[RTB]: Real-Time Bidding — the auction protocol where ad impressions are bought and sold in milliseconds.
*[CPM]: Cost Per Mille — the price paid per 1,000 ad impressions.
*[CTR]: Click-Through Rate — clicks divided by impressions, expressed as a percentage.
*[ROAS]: Return On Ad Spend — revenue generated per dollar spent on advertising.
*[BYOM]: Bring Your Own Model — Cat-Scan's system for plugging in your own external scoring model to drive optimization proposals.
*[CSV]: Comma-Separated Values — the file format Google Authorized Buyers uses for report exports.
*[AB]: Authorized Buyers — Google's programmatic ad exchange (formerly AdX / DoubleClick Ad Exchange).
*[HMAC]: Hash-based Message Authentication Code — a cryptographic signature used to verify webhook payloads.
*[SRE]: Site Reliability Engineering — the discipline of operating reliable production systems.
*[IAP]: Identity-Aware Proxy — Google Cloud's zero-trust access control for SSH and HTTP services.
*[DSN]: Data Source Name — the connection string used to connect to a PostgreSQL database.
